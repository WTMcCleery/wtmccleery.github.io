How to insert movie files into HTML5:

## Specify dimensions, add autoplay:
 <video width="320" height="240" autoplay>
 
## Give one or more versions of file for various browsers:
  <source src="movie.mp4" type="video/mp4">
  <source src="movie.ogg" type="video/ogg">
  
 ## Text to display if it doesn't work:
Your browser does not support the video tag.
</video>
